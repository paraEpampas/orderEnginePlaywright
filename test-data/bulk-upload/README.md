# Bulk Upload Test Data

Place Excel files here for the Bulk Upload tests. Each country needs its own file:

- `UK Bulk Upload.xlsx`
- `US Bulk Upload.xlsx`
- `DE Bulk Upload.xlsx`
- `BE Bulk Upload.xlsx`
- `NL Bulk Upload.xlsx`
- `FR Bulk Upload.xlsx`

Copy these from the original Java framework's Downloads folder or create them following the Order Engine bulk upload template format.

If a file is missing, the bulk upload test for that country will be skipped.
